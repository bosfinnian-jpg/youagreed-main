Design Rationale

Why This Exists

This installation visualises what happens after informed consent‚Äîdata fragments floating in a void, permanently exhibited. Unlike the anxiety-inducing countdown, this space is contemplative: a museum of cognitive extraction where visitors can explore the residue of human-AI conversation.

The Void as Metaphor

Data doesn't sit in folders‚Äîit exists in abstract space, replicated across servers, floating in cloud infrastructure. The 3D parallax void makes this abstraction tangible, letting visitors move through cognitive residue.

Depth as Permanence

Fragments exist at different depths‚Äîforeground, midground, background. This layering represents time: some data is recent and sharp, some fades into the distance but never disappears. All of it persists.

Mouse as Agency

Moving the mouse shifts perspective‚Äîvisitors have agency to explore, but cannot delete or remove fragments. This mirrors the reality of AI training data: you can look, but you cannot un-learn.

Development Notes

Process & Challenges

On Bruno Simon's Code

Experimentation & Limitations

I've been working with a stripped-down version of Bruno Simon's portfolio as a potential mainframe for my work. The code is extraordinarily powerful‚ÄîThree.js combined with Cannon.js physics creates experiences that feel genuinely tactile. However, the complexity is significant. The learning curve is steep, and adapting his approach to the conceptual needs of You Agreed. requires careful consideration of what's technically feasible within the project timeline.

On Simplicity

CSS as Constraint

This prototype uses pure CSS transforms and JavaScript for parallax‚Äîno WebGL, no Three.js. This is intentional constraint: before committing to complex 3D frameworks, I wanted to test whether the conceptual experience could emerge from simpler mechanics. The answer is partially yes‚Äîthe depth illusion works‚Äîbut the limitations are clear. True 3D navigation requires true 3D tools.

On Gimmickiness

Theatre vs. Substance

Some elements here feel theatrical‚Äîfragments floating in space, poetic labels like "The Void." This is acknowledged. The challenge is finding where spectacle serves meaning. Species in Pieces proves that visual drama can amplify conceptual weight; The Boat shows that immersion creates empathy. The question isn't whether to use theatre, but how to make it truthful.
